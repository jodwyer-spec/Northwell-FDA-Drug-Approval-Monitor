# Northwell FDA Drug Approval Monitor

Chrome extension that generates Excel reports of recently approved FDA drugs using the openFDA API.

## Features

- Download FDA drug approvals by date range
- Download 2-year master report
- Approval Type classification
- Specialty mapping
- Sponsor information
- Dosage form and route information
- Drug indication extraction
- ANDA filtering (generic approvals excluded)
- Multi-sheet Excel export

## Data Sources

- FDA Drugs@FDA API
- FDA Drug Label API

## Files

- manifest.json
- popup.html
- popup.js

## Installation

1. Download repository
2. Open Chrome
3. Navigate to chrome://extensions
4. Enable Developer Mode
5. Click Load Unpacked
6. Select project folder
